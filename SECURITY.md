# Security Policy

Mimir is pre-alpha. We are hardening the mesh before any commercial claims.

## Supported versions

None yet. We will publish a security policy with our first tagged release.

## Reporting a vulnerability

Please email security@mimir-mesh.dev (monitored by core maintainers) or open a private GitHub security advisory.

Do **not** file public issues for undisclosed vulnerabilities.

## Known open defects

Tracked in `ROADMAP.md` §24. All 🔴/🟠 defects must close before Gate G1.
